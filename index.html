How Websites Are Created

HTML

HTML, or "Hypertext Markup Language," is the standardized scripting language web browsers use to render pages. We will use HTML to form the skeleton or structure of our web page and add in elements such as(but not limited to) images and text. However, as you'll come to find out, there are some things basic HTML can't do, such as styling. Instead, to style our page, we will be using cascading style sheets or CSS.

CSS

Cascading Style Sheets, or CSS, is a style sheet language used to describe the presentation of a document written in a markup language such as HTML. Due to the visual limitations of basic HTML, although you can build functioning websites, it will almost always be used in conjunction with CSS. Whether it be changing font size or moving an element to the left side of a page, CSS can be used for all your styling and formatting needs.

JavaScript

JavaScript(or JS for short) is a scriping programming language used to develop interactive web pages and web applications. Although JS can be used alongside HTML stucture code and CSS styling code in web development the only one of the three that is a true "programming" language is JavaScript. WIth that, it is one of the most popular programming languages used today and can be used independantly or with JavaScript libraries to make super cool interactive websites and applications.

As you work more with these three languages everything will start to make more sense, but for now all you need to understand is HTML is used to define the contents of a web page, otherwise known as elements. CSS is used to style and specify the layout of web pages and lastly, Javascript is used to program the behavior of web pages. Successful Web Developers become comfortable with all three and utilize them to form user-friendly, responsive web pages.
