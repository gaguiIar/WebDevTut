HTML Structure

  1. <!DOCTYPE html>

    There are all kinds of files you can open on the web and each of them have properties that change the way they are displayed in a brower. Browsers need to be able to identify what kind if file is being loaded to figure out how to display it; the <!DOCTYPE html> tag allows us to do so. Writing <!DOCTYPE html> declares the document type as an HTML file and tells the browser how it should be displayed.

  2. <html>

    The html tag begins with <html> and ends with </html>. As we go through this tutorial you will see that most html tags have a starting and closing tag––Unless otherwise specified you can assume any future tags discussed in this tutorial have a closing tag that is identical to the primary tag with a forwardslash (/) in the beggining. In this case the beggining <html> tag identifies the start or root of a HTML document and the closing </html> tag indicates the end. Because the html tag is used to mark the start and end of an HTML document that means all other HTML elements present on your webpage will go inside these two tags.

  3. <head>

    The <head> tag defines elements about an html page such as the page title, links to CSS, external files, character encoding and other types of metadata about the document.

  4. <title>

    The <title> tag goes inside the head tag and defines the title of a webpage(Words that show up on webpage tab above url bar)

  5. <link>

    The <link> tag also goes inside the head tag and is used to import external resources such as CSS or cascading style sheet (.css) files.

  6. <body>

    The <body> tag will be the first HTML tag that proceeds the ending </head> tag. The <body> openeing and closing tags will contain eveyrthing that your browser will display.
