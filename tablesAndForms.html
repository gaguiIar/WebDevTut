Are you looking to break into tech but don't have extensive programming experience? My name is Gerardo, and I'm a sophomore studying Computer Science & Business Administration at the University of Southern California. I believe that with enough dedication, anyone can become a successful developer, so I built this website with that in mind. Introduction to Web Development is a resource aimed to help any beginner looking to branch into tech by breaking down web fundamentals such as HTML, CSS & JavaScript. If you're interested in learning more about web development, I applaud and encourage you to keep reading on. To start, you will want some type of IDE or Integrated Development Environment.     3. The html tag begins with <html> and ends with </html>. As we go through this tutorial you will see that most html tags have a starting and closing tag––Unless otherwise specified you can assume any future tags discussed in this tutorial have a closing tag that is identical to the primary tag with a forwardslash (/) in the beggining. In this case the beggining <html> tag identifies the start or root of a HTML document and the closing </html> tag indicates the end. Because the html tag is used to mark the start and end of an HTML document that means all other HTML elements present on your webpage will go inside these two tags.
