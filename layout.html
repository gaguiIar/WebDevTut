Layout

  1. float:

    Float is used to send an element to the left or right side of a page. At one point in time website layouts were made using the float property and invisible tables but now a days with the addition of flexbox and css grid, floats main purpose is to wrap text around images like so:

    floatExample.html

    <!DOCTYPE html>

    <html>

    <head>

    <title>This be the Verse</title>

    <link> rel="stylesheet" href="style/master.css">

    </head>

    <body>

    <img src = "imgs/pl.jpg" alt = "Philip Larkin">

    <p id="quote">Man hands on misery to man. It deepens like a coastal shelf. Get out as early as you can, And don’t have any kids yourself.<br>-Philip Larkin</p>

    </body>

    </html>




    master.css

    img {

    float: right;

    width: 200px;

    height: auto;

    }

    #quote {

    font-family: "ComfortaaReg", Arial, Sans Serif;

    padding: 20px;

    }

  2. Flexbox

    In the early stages of web development getting elements positioned right where you wanted them proved to be a huge task. In 2017 a new display type was intruced to CSS called flex which made page layout significantly easier.

    Flexbox uses css to effect the way parent containers and individual child flexitems react when it comes to things such as positioning, scaling and element wrapping.

    Elements inside the parent flexBoxContainer are called child flex items flexbox parent containers and items both have unique css properties and possible value combinations that do various things that effect the layout.

    To use flexbox you'll want to make a large "parent flex-container" You can do this by simply making a div container titled flexBoxContainer and then in CSS setting the display property to display: flex; like so:

    flexBoxExample.html

    <!DOCTYPE html>

    <html>

    <head>

    <title>Flex Example</title>

    <link rel="stylesheet" href="style/master.css">

    </head>

    <body>

    <div class = flexParentContainer>

    <div id = flexItem1>

    <p>hello</p>

    </div>

    <div id = flexItem2>

    <p>mate</p>

    </div>

    </div>

    </body>

    </html>



    master.css

    .flexParentContainer {

    display: flex;

    justify-content: SEE RIGHT;

    }

    #flexItem1 {

    color: green;

    }

    #flexItem2 {

    color: green;

    }
