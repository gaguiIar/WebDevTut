Body Tags

As the name implies HTML structure tags establish the foundation of any HTML webpage; However, to actually build the structure of a webpage you’ll want to deal with interal body tags as these are what is actually going to be rendered by your browser.

  1. <div>

    The <div> tag creates a division within your body. For example a navigation bar, photo gallery, paragraph, or biography can all be contained within a div container. DIv containers are useful for splitting your complex websites into digestible sections. Div containers can also be used with ids or classes to style specific groups of elements at once. We haven’t covered ID’s and Classes yet so don’t worry if that idea seems a bit complex, it will make more sense later.

  2. <p>

    The <p> tag is used to store text and has no special formating besides the fact that a <p> tag will automatically render a blank line before and after every <p> element

  3. <h#>

    Header, or <h#> tags can be used to make titles or of course headers within your webpage, Headers by default are bold in text and decrease or increase insize depending on how far the number used in the header tag is from 1, the first, principal heading tag. Possible tag variations are <h1>, <h2>, <h3>, <h4>, <h5>, <h6> with h1 being the largest and h6 being the smallest.

  4. <img>

    The img tag is used to add images to your website. For a image to be succesfully rendered you have to use it with the properties src and alt.

    src is used to priovide the location for the source of the image, if you have the image in your directory you can simply lead it to the file but alternatively you can use image urls by utalizing the url(“”) argument. If you will be using multiple photos its best you put them all in one central img file.

    alt is an important accessibility feature. The texts in the alt tag should desribe the image as it is what will be displayed if the image file cannot be found. Additionally for people who use screenreaders alt text will be used to describe what the image is. You should always have accurate and useful alt descriptors.

    Something to note img tags do NOT have a closing image tag.

  5. <a>

    The Anchor tag defnies a hyperlink to navigate from one page to another. You can use an anchor tag to direct someone to a already hosted url or a local html file like the img tag.

    Within the opening anchor tag you must utalize the href proeprty to link the website or file the hyperlink should navigate to.

    This is what a full anchor tag would look like:

    <body>

    <a href = "index.html"> Home Page <a>

  6. Lists:

    <ul>

    &

    <ol>

    HTML generally supports two types of lists, ordered and unordered.

    Ordered lists are listed using cardinal numbers and unordered lists are listed using bullet points. It is possible to have nested lists within main lists.

    The start of an ordered list can be triggered by using <ol> tags. The start of an unordered list can be triggered by using <ul> tags.

    Each list item must be contained within an opening <li> and closing </li> tag.

  7. <hr>

    &

    <br>

    The <hr> tag adds a thematic break and although it can be altered in CSS by default adds a horizontal black line where it is used. In this website two <hr> tags are used to define where the navigation bar starts and ends!

    The <br> tag manually inserts a line break that seperates text. <br> tags are particularly useful when writing addresses or poems.

    Note that neither the <hr> or <br> tag have closing tags.
