  2. Types of CSS

    There are three types of CSS–– External, Internal and inline. Now a days inline css is not standard and should not be used. Internal css on the other hand goes in the head of your html document under opening and closing <style> tags. Internal css only effects that html page. As the name implies external CSS is in an external file. The css in your css document will be applied to any html file that has that css file linked in the heading, as you recall we can link an external css in an html document by using a link tag. As I mentioned inline css is no longer standard and should not be used, so when should you use external over internal CSS? Well when you are building multi-length web-pagews and you want to have styling that is applied to them all this should be put in an external master css file as to avoid repeating code. If you want specific styling that will only be applied to one document add it internally in the head tag in between style tags.

    inlineCSSExample.html

    <!DOCTYPE html>

  • Useful CSS properties and values

color

h1 {

color: red;

}

makes text-color of h1 elements red

Other values: color, any hex code or RGB combination


background-color

h1 {

background-color: red;

}

sets background color red.

Other values: color, hex code or any RGB combination


background-image

h1 {

background-image: url('image.jpg');

}

sets background image as source provided

Other values: can be file or hosted link


display

h1 {

display: block;

}

changes how element is displayed

Other values: none, inline, block, inline-block


width

h1 {

width: 100px;

}

sets width of element

Other values: percentage, inherit, any pixel value or auto


height

h1 {

height: 100px;

}

sets height of element

Other values: percentage, inherit, any pixel value, auto


margin

h1 {

margin: 10px;

margin-left: 10px;

margin-right: 10px;

margin-top: 10px;

margin-bottom: 10px;

}

adds margin

Other values: any pixel value, inherit or auto


padding

h1 {

padding: 10px;

padding-left: 10px;

padding-right: 10px;

padding-top: 10px;

padding-bottom: 10px;

}

adds padding

Other values: any pixel value, inherit or auto


border

h1 {

border: 5px solid black;

border: double;

}

used to add border

Values define width, style and color in that order.

Other values: any pixel value; dotted, dashed; any color, hex or RGB value


font-family

h1 {

font-family: 'ComfortaaReg', Arial, Sans Serif;

}

defines font for element, will go from left to right if font can't be accessed

Other values: Sans Serif, Serif, Fancy and monospace are default font families.


font-size

h1 {

font-size: 10px;

}

defines font size

Other values: any pixel value, inherit or (x)em meaning inherited font emphasised x times

e.g.

2em = inherited font * 2


font-style

h1 {

font-style: normal;

}

defines font style

Other values: normal, italic, oblique
